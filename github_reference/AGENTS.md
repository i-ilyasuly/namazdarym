# Арнайы нұсқаулар (AI Agent Rules)

## 1. Құран аудио және мәтін синхрондау функциялары (Audio & Text Sync)
- **Файлдар:** `/src/context/QuranContext.tsx`, `/src/components/QuranVerseLive.tsx`, `/src/components/test-native/NativeQuranBlock.tsx`.
- **Ереже:** Бұл файлдардағы аудио синхрондаудың "айнымалы математикалық қозғалтқышы" (smooth predictive interpolation, 60fps) және 1-10 қаріп өлшемдеріне (font size scale) жауапты логикалар мінсіз жұмыс істеп тұр. 
- **Қатаң ескерту:** Бұл логикаға **ТЕК** қолданушының өзі тікелей тапсырма беріп, жаңа Мұсхафтар немесе қаріптер (шрифттер) қосу керек екенін айтқан жағдайда ғана өте абайлап өзгеріс енгіз. Ешқандай нұсқаусыз бұл алгоритмдерге жанасуға, бұзуға немесе оңайлатуға рұқсат етілмейді. Өзгерту қажет болғанда кадрлық жиілік (60fps) пен мәтіннің жұмсақ ағуын жоғалтып алмау қадағалансын.

## 2. Терминология және Навигация (Terminology & Navigation)
- **Басты бет (Main Page):** Веб-қосымшаның (Browser) негізгі беті.
- **Намазым беті (Namazym Page):** Тест (Mobile Prototype) бөліміндегі басты бет (намаз кестесі мен құран ағыны бар бет).
- **Бағдар беті (Bagdar Page):** Тест бөліміндегі статистика беті.
- **Ереже:** Қолданушы "басты бет" десе веб-қосымшаны, "намазым беті" десе тест бөлімін меңзеп тұрғанын есте сақта. Тест бөліміндегі навигацияны осы терминдерге сай сақта.
