# Momyn App: Даму Жоспары

Осы файлда қолданушының сұрақтары мен жауаптарына негізделген алдағы жұмыстар тізімі мен жоспары көрсетілген.

## 1-кезең: Негізгі функционал және Firebase (Орындалуда)
- [x] Firebase жобасын баптау (Auth және Firestore).
- [x] Профиль бетінде Google арқылы кіру/шығуды енгізу.
- [x] Намаз қадағалау блогын жасау (Status tracking).
- [x] Намаз мәртебесін Firestore-ға сақтау (Alone, Jamaat, Late, Qaza).
- [x] Намазға кешігу немесе оқымау себептерін (Reasons) таңдау мүмкіндігі.
- [ ] 48 сағат ішінде деректерді өңдеу (Edit) мүмкіндігін қосу.
- [ ] Күнтізбе (Calendar) бөлімін толықтай іске қосу.

## 2-кезең: Статистика және Талдау
- [ ] Firestore-дағы нақты деректер негізінде `StatsScreen` статистикасын шығару.
- [ ] Әр намаз үшін жеке фильтр (Таң, Бесін т.б.) арқылы статистика көру.
- [ ] Белсенділік картасын (Heatmap) нақты деректермен байланыстыру.
- [ ] Апталық, айлық және жылдық есептерді дайындау.

## 3-кезең: Хабарламалар және Қосымша мүмкіндіктер
- [ ] Күнделікті сағат 21:00-де намаздарды белгілеуді еске салатын хабарлама (Notification).
- [ ] Қолданушы параметрлерін (Theme, Prayer times settings) Firestore-да сақтау.
- [ ] Намаз уақыттарының алгоритмін нақтылау (Кешігу уақытын анықтау үшін).
- [ ] Түнгі/Күндізгі режим және түстер фильтрінің барлық экрандарда дұрыс жұмыс істеуін қадағалау.

## 4-кезең: Әлеуметтік және дамыту (Болашақта)
- [ ] Достар қосу және олармен нәтижелерді салыстыру.
- [ ] Leaderboard (Рейтинг) жүйесі.
- [ ] Көптілділік (Multilingual support).
- [ ] Түнгі оқу немесе қосымша намаздарды қадағалау.

---
**Есте сақтау керек нәрселер:**
- Намаз блоктарындағы 4 иконка статус болып табылады.
- Статистика бетіндегі түстер таңдалған тақырыпқа (Theme) сай болуы тиіс.
- Қолданушыға түсінікті болуы үшін арнайы "Туториал" (нұсқаулық) қажет.
